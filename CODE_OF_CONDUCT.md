# Code of Conduct

## Our Standards

We are committed to providing a welcoming and professional environment for all contributors. We expect participants to:

- Use welcoming and inclusive language
- Be respectful of differing viewpoints and experiences
- Accept constructive criticism gracefully
- Focus on what is best for the community and project
- Show empathy towards other community members

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, and any other communication channels.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior. They may remove, edit, or reject contributions that do not align with this Code of Conduct.

## Attribution

This Code of Conduct is adapted from common open source community standards.
