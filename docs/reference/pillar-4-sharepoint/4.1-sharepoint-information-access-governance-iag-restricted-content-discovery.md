# Control 4.1: SharePoint Information Access Governance (IAG) / Restricted Content Discovery

## Overview

**Control ID:** 4.1  
**Control Name:** SharePoint Information Access Governance (IAG) / Restricted Content Discovery  
**Regulatory Reference:** GLBA 501(b), 504(b), SEC Reg S-P, FINRA 4511  
**Setup Time:** 1-2 hours  

---

## Governance Levels

### Baseline (Level 1)

Enable Restricted Content Discovery (RCD); limit agent access to approved sites.

### Recommended (Level 2-3)

IAG policies restricting access by department/tier; quarterly access reviews.

### Regulated/High-Risk (Level 4)

Mandatory RCD for Zone 3; admin-controlled access lists; immutable access logs.

---

## Verification & Testing

1. SharePoint Admin Center → Policies → Restricted Content Discovery
2. Verify RCD is enabled
3. Check approved site list for agent access
4. Verify access audit logs
EXPECTED: RCD enabled; agent access restricted to approved sites
