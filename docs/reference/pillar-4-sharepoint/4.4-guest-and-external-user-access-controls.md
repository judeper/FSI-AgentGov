# Control 4.4: Guest and External User Access Controls

## Overview

**Control ID:** 4.4  
**Control Name:** Guest and External User Access Controls  
**Regulatory Reference:** GLBA 501(b), 504(b), SEC Reg S-P, FINRA 4511  
**Setup Time:** 30-45 min  

---

## Governance Levels

### Baseline (Level 1)

Restrict guest sharing; disable external user access for sensitive sites.

### Recommended (Level 2-3)

Guest sharing disabled for Zone 3 sites; time-limited access for Zone 2.

### Regulated/High-Risk (Level 4)

No external access to regulated content; guest access audit trail.

---

## Verification & Testing

1. SharePoint site → Settings → External sharing → Sharing settings
2. Verify external sharing is disabled or restricted
3. Check for any guest users and validate their access
4. Review sharing audit log
EXPECTED: External access controlled; guest users restricted or audited
