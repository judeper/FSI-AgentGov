# Control 2.2: Environment Groups and Tier Classification

## Overview

**Control ID:** 2.2  
**Control Name:** Environment Groups and Tier Classification  
**Regulatory Reference:** FINRA 4511, GLBA 501(b), SOX 302/404  
**Setup Time:** 30-60 min  

---

## Governance Levels

### Baseline (Level 1)

Classify environments by tier (Development, Test, Production); segregate by governance risk.

### Recommended (Level 2-3)

Environment groups with tier-specific governance policies; documented tier definitions.

### Regulated/High-Risk (Level 4)

Strict segregation; separate admins per tier; tier changes require documented approval.

---

## Verification & Testing

1. Power Platform Admin Center → Manage → Environment groups

3. Confirm governance policies differ by tier
4. Test policy enforcement per tier
