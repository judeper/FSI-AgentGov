# Control 1.16: Information Rights Management (IRM) for Documents

## Overview

**Control ID:** 1.16  
**Control Name:** Information Rights Management (IRM) for Documents  
**Regulatory Reference:** GLBA 501(b), SEC Reg S-P, FINRA 4511, SOX 404  
**Setup Time:** 30-45 min  

---

## Governance Levels

### Baseline (Level 1)

Apply IRM to sensitive documents; restrict copy, print, screenshot, forward permissions.

### Recommended (Level 2-3)

IRM applied to all Zone 2+ documents; time-limited access; audit logging of document access.

### Regulated/High-Risk (Level 4)

Mandatory IRM with expiration; no download without audit; continuous access revocation capability.

---

## Verification & Testing

1. SharePoint library or OneDrive → Select sensitive document
2. Apply IRM protection (File → Info → Protect Document)
3. Verify permissions: No print, no copy, no screenshot
4. Share link and test access restrictions
EXPECTED: IRM prevents unauthorized document use; permissions enforced
