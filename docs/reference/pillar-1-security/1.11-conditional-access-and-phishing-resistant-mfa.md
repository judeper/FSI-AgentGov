# Control 1.11: Conditional Access and Phishing-Resistant MFA

## Overview

**Control ID:** 1.11  
**Control Name:** Conditional Access and Phishing-Resistant MFA  
**Regulatory Reference:** GLBA 501(b), FINRA 4511, SEC Reg SHO, SOX 302  
**Setup Time:** 1-2 hours  

---

## Governance Levels

### Baseline (Level 1)

Require MFA for all agent creators and publishers; document MFA policy.

### Recommended (Level 2-3)

Conditional Access policies based on risk; enforce phishing-resistant authentication for Zone 2/3.

### Regulated/High-Risk (Level 4)

Mandatory phishing-resistant MFA (FIDO2/Windows Hello) for Zone 3; risk-based policies with continuous evaluation.
