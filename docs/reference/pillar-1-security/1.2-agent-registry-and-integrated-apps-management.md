# Control 1.2: Agent Registry and Integrated Apps Management

## Overview

**Control ID:** 1.2  
**Control Name:** Agent Registry and Integrated Apps Management  
**Regulatory Reference:** FINRA 4511, SEC Rule 17a-3/4  
**Setup Time:** 30-60 min  

---

## Governance Levels

### Baseline (Level 1)

Centralized inventory (spreadsheet) of agents, updated at least monthly.

### Recommended (Level 2-3)

Automated registry with ownership, data sources, connectors, approval status; weekly review.

### Regulated/High-Risk (Level 4)

Real-time inventory with automated drift detection; daily monitoring for production agents.

---

## Verification & Testing

1. M365 Admin Center → Settings → Integrated Apps
2. Verify list includes all agents
3. Confirm metadata: Owner, Zone, Data Sources
4. Export inventory (CSV/Excel)
EXPECTED: Complete, current inventory with all required fields
