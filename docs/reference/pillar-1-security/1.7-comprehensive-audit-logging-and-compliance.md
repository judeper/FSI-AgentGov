# Control 1.7: Comprehensive Audit Logging and Compliance

## Overview

**Control ID:** 1.7  
**Control Name:** Comprehensive Audit Logging and Compliance  
**Regulatory Reference:** FINRA 4511, SEC 17a-3/4, SOX 302/404, GLBA 501(b)  
**Setup Time:** 1-2 hours (Standard) or 8-10 hours (Azure WORM)  

---

## Governance Levels

### Baseline (Level 1)

Purview Audit Standard logging + defined review cadence; export key events externally if retention insufficient.

### Recommended (Level 2-3)

Purview Premium (1+ years) + audit retention policies; SIEM integration for Zone 2/3 agents; export weekly.

### Regulated/High-Risk (Level 4)

Purview Premium + 10+ year retention OR weekly export to Azure immutable storage; continuous anomaly detection.

---

## Verification & Testing

1. Microsoft Purview → Audit → Search
2. Search for agent/Copilot events in past 24 hours
3. For Zone 3: Confirm retention policy set to 10 years
4. Verify immutable storage configured
EXPECTED: Audit logs current; retention configured per regulation; immutable archive in place
