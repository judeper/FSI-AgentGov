# Control 3.6: Orphaned Agent Detection and Remediation

## Overview

**Control ID:** 3.6  
**Control Name:** Orphaned Agent Detection and Remediation  
**Regulatory Reference:** FINRA 4511, SOX 404, GLBA 501(b)  
**Setup Time:** 30-45 min  

---

## Governance Levels

### Baseline (Level 1)

Identify agents without assigned owner; document remediation plan.

### Recommended (Level 2-3)

Automated detection; monthly review; email notifications to IT.

### Regulated/High-Risk (Level 4)

Real-time detection; automatic suspension until owner assigned.

---

## Verification & Testing

1. Run orphaned agent report (export from M365 Admin Center)
2. Verify any orphaned agents are listed
3. Check remediation actions (reassign or decommission)
4. Confirm follow-up is scheduled
EXPECTED: Orphaned agents detected; remediation planned or completed
