# Validation Testing

This section provides guides for validating governance controls, scripts, and configurations.

---

## Available Guides

| Guide | Purpose |
|-------|---------|
| [Script Validation Guide](script-validation-guide.md) | Validate PowerShell and KQL scripts used in playbooks |

---

## When to Use These Guides

- **Before Production Deployment:** Validate all scripts work in your environment
- **After Module Updates:** Re-validate scripts when PowerShell modules are updated
- **During Troubleshooting:** Diagnose script failures and connectivity issues
- **For Audit Preparation:** Document validation evidence for regulatory examinations
