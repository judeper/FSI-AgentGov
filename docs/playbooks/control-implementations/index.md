# Control Implementation Playbooks

Implementation playbooks provide step-by-step portal walkthroughs, PowerShell scripts, verification procedures, and troubleshooting guidance for each control in the FSI Agent Governance Framework.

---

## How to Use These Playbooks

Each control in the [Control Catalog](../../controls/index.md) links to four implementation playbooks:

| Playbook | Purpose |
|----------|---------|
| **Portal Walkthrough** | Step-by-step configuration using Microsoft admin portals |
| **PowerShell Setup** | Scripts for automation and bulk configuration |
| **Verification & Testing** | Test cases, evidence collection, and validation procedures |
| **Troubleshooting** | Common issues, diagnostic steps, and resolutions |

### Navigation

Navigate to any control in the Control Catalog to find links to its implementation playbooks in the **Implementation Guides** section.

## Playbook Format

Each playbook follows a consistent structure:

### Portal Walkthrough
- Prerequisites checklist
- Step-by-step portal configuration
- Configuration by governance level
- Screenshots and UI references
- Validation checkpoints

### PowerShell Setup
- Module requirements
- Configuration scripts with full documentation
- Validation scripts
- Complete automation examples

### Verification & Testing
- Manual verification steps
- Test case table
- Evidence collection checklist
- Automated validation scripts

### Troubleshooting
- Common issues table (Issue | Cause | Resolution)
- Detailed diagnostic steps
- Configuration confirmation methods
- Escalation path
