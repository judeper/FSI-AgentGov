# Templates

This folder contains assessment and policy templates for the FSI Agent Governance Framework.

## Available Templates

| Template | Description | Format |
|----------|-------------|--------|
| `FSI_Agent_Governance_Framework_v1.0_Beta.xlsx` | Comprehensive assessment workbook with 22 sheets covering all 43 controls, maturity scoring, and compliance tracking | Excel |
| `FSI_Agent_Governance_Complete_v1.0_Beta.docx` | Complete policy document template with all 43 control sections | Word |

## How to Use

### Assessment Workbook (Excel)

The Excel workbook provides comprehensive assessment and tracking capabilities:

1. **Dashboard** - Executive overview of compliance status
2. **Self-Assessment Sheets** - Complete assessments for each control (Pillars 1-4)
3. **Compliance Matrix** - Track regulatory alignment by regulation
4. **Implementation Checklist** - Deployment planning and tracking
5. **Maturity Scoring** - Score controls at levels 0-4

**Getting Started:**
1. Start with the Dashboard sheet for an overview
2. Complete self-assessments for each pillar
3. Use the Compliance Matrix to track regulatory alignment
4. Review the Implementation Checklist for deployment planning

### Policy Document (Word)

The Word document provides a complete policy template:

1. **43 Control Sections** - One section per control with customizable guidance
2. **Executive Summary** - High-level policy overview
3. **Appendices** - Supporting reference materials

**Getting Started:**
1. Customize the template for your organization
2. Document your specific implementation decisions
3. Use as the basis for governance committee approval
4. Update quarterly as controls mature
